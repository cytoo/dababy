## Installation

first of all you'll need cargo (the rust package manager)

install cargo with:

`curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh`

after that run the following command:

`cargo install dababy`

## Commands
`dababy <message>`

`-b` for blue dababy
